Lab 11 - JavaScript Events and Forms

Challenge

The challenge of this lab is to experiment with JavaScript events by taking user input from a form field, modifying it using JavaScript and displaying the result using jQuery.

Problems

The main problem we had in doing this lab was actually having the functions show up in the webpage. We debugged it by fixing the structure and syntax of the code.
